However, I did not deal with the html portion nor that proficient at it. Personally, I am more of a looking into data person, however I hope after this project, I can see how the CRUD(Create, Read, Update, Delete) idea can be taken in place.

With this stated, I recently went to Halloween Horror Nights, and I wanted to do some more research on theme park events such as New Years, Halloween, or Christmas. This would allow me to compare and contrast some theme parks from another. One example being Universal and Disney(because I am not the biggest Disney fan.)

      For this project, since my html experience is not necessarily the best, I was able to follow the rubric to create this website. I utilized html code from Visual Studio Code as well as utilize CSS as a format for my pages. Then, for my analytics page I was able to extract some Java code to create a visualization on theme park reviews. Overall, I think this project was a good way to get my feet into html.
